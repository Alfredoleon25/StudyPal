# STUDYPAL

A simple web application to connect learners with tutors, built with **TypeScript, React, Prisma, and PostgreSQL**.

---

## Features

- Browse tutors based on your subjects
- Request help and start chats with tutors
- View your chat history as a learner or tutor
- Manage your subjects
- Clean and responsive interface

---

## Tech Stack

- **Frontend:** React + TypeScript
- **Backend:** Node.js + Prisma + PostgreSQL
- **Database:** PostgreSQL
- **Styling:** Inline styles and CSS

---

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/yourusername/tutor-chat-app.git
cd tutor-chat-app

2. **Install dependencies**

Frontend:
cd frontend
npm install


cd backend
npm install

3 **Configure Databasee**

Create a PostgreSQL database
Update .env in the backend folder with your database credentials

4 **Run the application**

Backedend: npx nodemon index.js
frontend: npm run dev
